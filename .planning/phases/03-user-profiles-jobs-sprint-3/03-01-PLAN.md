# Phase 3, Plan 1: User Profiles (HIQ-021 to HIQ-023)

**Target Phase:** 03-user-profiles-jobs-sprint-3
**Goal:** Build GET/PUT `/users/profile` endpoints, profile completion calculator, and `POST /users/avatar` direct S3 upload logic.

## Technical Context
- **Dependencies:** 02-01-PLAN.md (User table/model exists)
- **Key Constraints:** `aioboto3` direct stream upload. Profile completion logic implemented dynamically via Pydantic `@computed_field`.
- **New files:** `backend/app/api/v1/users.py`, `backend/app/schemas/user.py`, `backend/app/core/s3.py`, `backend/app/models/profile.py`
- **Modify:** `backend/app/main.py` (register users router), `backend/app/core/config.py` (add S3 env vars)

## Action Plan

### 1. S3 Config & Core setup (`backend/app/core/config.py` & `s3.py`)
- [ ] Add `AWS_ACCESS_KEY_ID`, `AWS_SECRET_ACCESS_KEY`, `AWS_REGION`, and `S3_BUCKET_NAME` to `Settings` in `config.py`.
- [ ] Create `backend/app/core/s3.py`:
  - Provide a function `async def get_s3_client()` that yields an `aioboto3.Session().client('s3', config=Config(signature_version='s3v4'))`.
  - Provide an upload helper `async def upload_avatar(file: UploadFile, user_id: UUID) -> str` that reads the file, validates size (<2MB) and MIME type (`image/jpeg`, `image/png`), and uses `put_object` to upload to S3 at `avatars/{user_id}.{ext}`. Returns the S3 key.

### 2. Database Models (`backend/app/models/profile.py`)
- [ ] Create `Profile` SQLAlchemy ORM model.
  - `id`: ForeignKey to `users.id`, primary_key=True.
  - `bio`: Text, nullable.
  - `avatar_url`: String(512), nullable.
  - `skills`: ARRAY(String), server_default='{}'.
  - Establish relationship with `User` model (`uselist=False`).
- [ ] Add `profile = relationship("Profile", back_populates="user")` to the `User` model.

### 3. Schemas (`backend/app/schemas/user.py`)
- [ ] Create `ProfileUpdate(BaseModel)`: `bio: Optional[str]`, `skills: Optional[List[str]]`.
- [ ] Create `ProfileResponse(BaseModel)`:
  - Fields: `bio`, `avatar_url`, `skills`.
  - Add `@computed_field` for `completion_percentage` (avatar=20%, bio=30%, skills=50%).

### 4. API Endpoints (`backend/app/api/v1/users.py`)
- [ ] Create router with `prefix="/users"`, `tags=["Users"]`.
- [ ] `GET /profile`:
  - Fetch profile for `current_user`. If none exists, return empty profile (0%).
- [ ] `PUT /profile`:
  - Update or create profile record with `ProfileUpdate` data.
- [ ] `POST /avatar`:
  - Accept `UploadFile`. Use `upload_avatar` helper.
  - Update `Profile.avatar_url`. Return new URL.

### 5. Integration
- [ ] Add `uvicorn` startup logic to create DB tables (or use Alembic).
- [ ] Register `users_router` in `backend/app/main.py`.
- [ ] Add `aioboto3` to `requirements.txt`.

## Verification
- [ ] `GET /v1/users/profile` returns `completion_percentage: 0` for new user.
- [ ] `PUT /v1/users/profile` with skills and bio returns `completion_percentage: 80`.
- [ ] `POST /v1/users/avatar` successfully uploads to S3 (mocked or real) and returns `completion_percentage: 100`.
